== Description ==

Introducing ConvertPro.

Welcome to ConvertPro's A/B Tester for Pages Plugin! Enhance your website's performance by experimenting with different page variations effortlessly. Test, analyze, and optimize your pages to maximize conversion rates and achieve your goals effectively. Get started with ConvertPro's intuitive A/B testing features today!


== Installation ==

1. Upload the `convert-pro` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
